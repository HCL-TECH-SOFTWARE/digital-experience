# Using DAM
Upload, manage, and modify rich media assets and collections using HCL Digital Asset Management (DAM).

- **[Accessing DAM](./access_digital_asset_mgmt.md)**  
  Learn how to access HCL DAM and use it as a central platform to store and include rich media assets.  

- **[Managing DAM](./managing_dam/index.md)**  
  Learn about managing, uploading, importing, exporting, and modifying your DAM media assets.  

- **[Searching in DAM](./search_dam.md)**  
  Learn how to search for the right DAM asset.  

- **[Using the DX Content Picker for DAM](./dxpicker.md)**  
  Learn how to search for and select a DAM media asset for use in any application.
