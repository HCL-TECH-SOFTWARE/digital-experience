# Integration

This section includes how-to articles about integration-related content in HCL Digital Experience (DX).

- **[How to integrate Leap and DX so they can be accessed from the same domain](./how-to-integrate-leap.md)**  
This guides shows you how to install Leap and DX on the same cluster and set up Ingress or Gateway API so Leap and DX can be accessed from the same domain.

- **[How to setup LTPA SSO between DX and Leap](./enable-sso-leap-dx.md)**  
This guides shows you how to set up Lightweight Third Party Authentication Single Sign On between Leap and DX so that logging in on one app securely logs you in to the other without having to re-enter credentials.

- **[How to Embed a Leap application into a DX page](./embed-leap-app.md)**  
This guides shows you how to embed a Leap application into a DX page. Take note that before doing this, you should first be able to both DX and Leap on the same domain, and have set up SSO.
