# How to integrate HCL DX with LDAP

## Applies to

> HCL Digital Experience v8.5 and higher

## Introduction

Applications commonly require HCL Digital Experience (DX) to integrate with existing user repositories to enable authentication, authorization, and user management. LDAP (Lightweight Directory Access Protocol) is the most common user repository type.

In most organizations, different groups administer HCL DX and the LDAP server. Integrating the two requires in-depth knowledge of both systems. This articles provides you with the necessary information to integrate HCL DX with existing LDAP servers.
