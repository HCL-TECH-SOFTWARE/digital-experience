# How to install and configure an HCL Digital Experience v9.5 Cluster

## Applies to

> HCL Digital Experience 8.5 and higher

## Introduction

This guide describes a comprehensive procedure for installing, configuring, and building an HCL Digital Experience (HCL Portal) v9.5 cluster using:  

- Red Hat Enterprise Linux 7.7
- IBM WebSphere Application Server (WAS) 9.0.5 – 64 bit
- DB2 11.1 – Database Server
- IBM Directory Server 6.4 – LDAP Server
- IBM HTTP Server 9.0.5 – Web Server
