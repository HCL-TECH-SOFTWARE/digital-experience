# How to install and configure an HCL DX 9.5 cluster

## Applies to

> HCL Digital Experience 8.5 and higher

## Introduction

This article provides a step-by-step guide for installing, configuring, and building a highly available and scalable HCL Digital Experience (DX) 9.5 cluster. It covers the essential components to ensure a successful deployment, including the operating system, application server, database, and directory server. This guide uses the following software:  

- DB2 11.1 – Database Server
- IBM Directory Server 6.4 – LDAP Server
- IBM HTTP Server 9.0.5 – Web Server
- IBM WebSphere Application Server (WAS) 9.0.5 – 64 bit
- Red Hat Enterprise Linux 7.7
